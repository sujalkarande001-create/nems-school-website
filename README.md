# New English Medium School Website

Professional static website for **New English Medium School, Sambhaji Nagar, Kolhapur**.

## Files

- `index.html` - complete school website
- `styles.css` - responsive design and layout
- `script.js` - mobile menu and admission enquiry interaction
- `assets/` - school logo and board photo

## Deploy

Upload all files to any static hosting provider such as GitHub Pages, Netlify, Vercel, Hostinger or cPanel public HTML.
